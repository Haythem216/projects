# Blender scene

A 3D scene built in Blender, with Python scripting for terrain, structures, and materials.

**FinalProject.mp4** is a recording of the scene (render/animation of the blend file).

## Contents

| File | Description |
|------|-------------|
| `FinalProject.blend` | Blender scene file |
| `FinalProject.py` | Python script (Blender API) used to build the scene |
| `FinalProject.mp4` | Recording of the scene |
| `BoatModel.FBX` | Boat model asset |
| `*.jpg`, `*.png` | Textures (bark, brick, roof, etc.) |

## Tech

- Blender, Python (Blender API)
