# Tunisia — Touristic Destination

A multi-page responsive website advertising Tunisia as a touristic destination. Covers major cities (Hammamet, Mahdia, Tozeur), geography, culture, and travel info with media, maps, and video.

**Live site:** [https://bonsai.lcsc.edu/hbaccar/CITPT-111/FinalProj/](https://bonsai.lcsc.edu/hbaccar/CITPT-111/FinalProj/)

## Features

- **Home** — Intro, map, and overview of Tunisia
- **Hammamet** — "Tunisian Saint-Tropez": landscapes, climate, beaches
- **Mahdia** — Coastal town, history, Fatimid heritage, beaches
- **Tozeur** — Gateway to the Sahara, oasis, desert, palm groves
- **Average Cost** — Travel cost info
- Responsive layout, navigation, media (images, video)

## Structure

| Path | Description |
|------|-------------|
| `FinalProj/index.html` | Home |
| `FinalProj/hammamet.html`, `mahdia.html`, `tozeur.html` | City pages |
| `FinalProj/cost.html` | Cost page |
| `FinalProj/css/styles.css` | Styles |
| `FinalProj/javaScript/popup&nav.js` | Navigation & popups |
| `FinalProj/media/` | Images, video (e.g. Tunisian-Paradise) |
